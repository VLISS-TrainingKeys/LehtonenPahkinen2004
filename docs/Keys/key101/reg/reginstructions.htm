INSTRUCTIONS FOR TRAINING KEY 101a PART B: REGRESSION ESTIMATION WITH DIFFERENT SAMPLES

In this exercise we will study further the regression estimation of totals by using the Province'91 data as the frame population. Regression estimation will be applied for the total of UE91 (the number of unemployed in a county in 1991) by using one auxiliary variable HOU85 (the number of households according to the population census 1985). We will select different samples to examine the effect of the sample size and the sample configuration on the estimates. Regression estimates are calculated by the SAS procedure SURVEYREG.

HOW TO MANAGE THIS EXERCISE

1) View the frame population from the link on the left frame to see the original population and the population correlations of UE91 and HOU85.

2) Select the sample size (n=8,12 or 16) and sample configuration* (sample 1 or sample 2) from the drop-down menu on the left and view the results by pressing submit button. In each case, SRSWOR estimate for UE91 total  can be found from data summary table as weighted sum of UE91 and regression estimate and its standard error estimate from analysis of estimable functions table  - UE91 total.

3) After viewing the results, you can display summary table from the dropdown menu to compare the results for each selected sample.

NOTE. The population total of UE91 is 15098 persons. Use this figure as a reference when comparing the results.

*) Sample configuration is determined by the seed number for the random number generator.
