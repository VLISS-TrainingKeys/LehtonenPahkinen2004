INSTRUCTIONS FOR KEY 277 PART B: LOGIT ANOVA WITH THE GWLS METHOD

In Part B we demonstrate how to fit a logit ANOVA model for the OHC survey data with the GWLS (Generalized weighted least squares) method. The purpose of this exercise is to give a more technical view on the GWLS method, because the necessary formulas are programmed with a matrix language. Numerical results of Example 8.1 of Lehtonen and Pahkinen (2004) also will be reproduced.

The GWLS method of generalized weighted least squares estimation provides a flexible technique for ANOVA type logit modelling in the case where all predictor variables are categorical. GWLS estimation can be used under the full design-based DES option and all the complexities of the sampling design including stratification, clustering and weighting, will be allowed. Analysis under the BIN option assuming simple random sampling serves as a reference, because all the sampling complexities will be ignored under this option.

The GWLS method is computationally simple because closed-form expressions can be used (requiring no iteration). The execution of the GWLS method is programmed in SAS/IML language (Interactive Matrix Language).

2) Principles in fitting the logit ANOVA model with the GWLS method

We apply the GWLS method to fit the main effects model:

LOGIT (PSYCH)= INTERCEPT + SEX + AGE + PHYS

When applying the GWLS method for logit modelling under a given analysis option, DES or BIN, the starting point is the calculation of the corresponding proportion estimate vector and its covariance matrix estimate. By using these estimates, the model coefficients are estimated, together with the covariance matrix of estimated coefficients. Using these estimates, fitted proportions and their covariance matrix estimate is obtained. Further, the Wald test of goodness of fit of the model, and desired Wald tests of linear hypotheses on the model coefficients, is executed. Finally, residual analysis is carried out to more closely examine the fit of the selected model.

Please note that the GWLS method applies for both DES and BIN options. The key point is the selection of the proportion estimator vector and its covariance matrix estimator, which properly reflect the sampling design assumption.
