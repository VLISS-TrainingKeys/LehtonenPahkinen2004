KEY 288: Logistic ANCOVA

INTERPRETATION OF MODEL 3

MODEL 3

logit(PSYCH) = SEX + AGE + PHYS + CHRON

Model 3 includes only the main effect terms. Please note the following points:

1. In Model 3, all interactions have been removed. This is just to show how a main effects model would look like. Interpretation of Model 3 is quite clear but can end up to inadequate conclusions.

2. Graphs show that the only source of variation is due to the inclusion of all the main effects in the model. Again, the fitted proportions of falling into the high psychic strain group increase with increasing age, for both sexes. Also, in females, fitted proportions tend to be larger than for males. But the lines are parallel for males and females in all graphs. This can cause an incorrect interpretation of the phenomenon under study.
