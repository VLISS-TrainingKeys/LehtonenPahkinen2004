INSTRUCTIONS FOR KEY 288: Logistic ANCOVA by SUDAAN/RLOGIST

The purpose in this exercise is to demonstrate the role of interaction terms in design-based multivariate logistic ANCOVA modelling. The OHC Survey data set will be used. The binary response variable PSYCH2 measures psychic strain (1: present, 2: otherwise). AGE (in years), PHYS (binary; Physical health hazards; 1 present, 0 otherwise) and CHRON (binary; Chronic morbidity; 1 present, 0 otherwise) are treated as continuous predictors. SEX is the discrete predictor. Thus there are four predictors available. Because SEX is the only discrete predictor, interaction terms of SEX with AGE, PHYS and CHRON can be included in this model setting (note that we will not include interactions formed by continuous predictors).

HOW TO MANAGE THIS EXERCISE

Please examine the different modes by selecting a model (Model1, Model 2 or Model 3) from the SELECT MODEL drop-down menu. The first model includes all the two-variable interaction terms of SEX with AGE, PHYS and CHRON. Please examine the p-values of t-tests to determine whether an interaction term is statistically significant. Model 2 includes one interaction term, which appears to be statistically significant at 5 % level. Model 3 includes only the main effects. For each model, also the odds ratio statistics (OR) and their standard errors are calculated.

For each model, predicted proportions of PSYCH2 are calculated and presented graphically, for each PHYS-CHRON combination.  Pay careful attention to these graphs, because they appear to be very useful in interpreting the models and the role of the interaction terms. Try to explain the results from each model.

Please compare the results with the logit ANOVA exercise (KEY 277). What conclusions can be drawn?

MODEL 1:  LOGIT(PSHYCH2) = INTERCEPT + SEX + AGE + PHYS + CHRON + SEX*AGE + PHYS*SEX + CHRON*SEX

MODEL 2:  LOGIT(PSYCH2) = INTERCEPT + SEX + AGE + PHYS + CHRON + SEX*AGE

MODEL 3: LOGIT(PSYCH2) = INTERCEPT + SEX + AGE + PHYS + CHRON

Guidelines for the interpretation of each model are given in the bottom drop-down INTERPRETATION HELP menu.
