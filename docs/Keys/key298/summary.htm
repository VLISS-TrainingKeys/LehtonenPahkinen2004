SUMMARY OF RESULTS
Continuous study variable PSYCH

Analysis ignoring the clustering effect by SAS procedure REG involves overly liberal testing result for AGE effect. The analysis suggests a model that is too complex. Ignoring the clustering effect can lead to erroneous inferences.

Analyses accounting properly the clustering effect support a simpler model where AGE effect is excluded. GEE analyses with SUDAAN/REGRESS and SAS/GENMOD are closely comparable with each other and with REML-based analysis for a mixed model with SAS/MIXED.
