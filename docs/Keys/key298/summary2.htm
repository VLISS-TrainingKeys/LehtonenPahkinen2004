SUMMARY OF RESULTS
Binary variable PSYCH2
(1=more severe psychic strain, 0=less severe psychic strain)

Analysis ignoring the clustering effect by SAS procedure GENMOD involves overly liberal testing result for AGE effect. Analyses accounting properly the clustering effect support a similar model, but significance of AGE effect is weaker when compared to the ML-based analysis. GEE analyses with SUDAAN/LOGISTIC(RLOGIST),SAS/GENMOD and R/gee, assuming an exchangeable correlation structure, are closely comparable with each other and with REML-based analysis for a mixed model with SAS macro GLIMMIX and R function glmmPQL.
