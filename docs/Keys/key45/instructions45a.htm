INSTRUCTIONS FOR TRAINING KEY 45 PART A: INTRA-CLASS CORRELATION FOR SYSTEMATIC SAMPLING IN THE PROVINCE'91 POPULATION

In this exercise we calculate the intra-class correlation coefficient under systematic sampling. The connection between the intra-class correlation coefficient and design effect DEFF is also demonstrated. We use the Province'91 data set as the frame population. Results of Example 2.4 will be reproduced. We also provide a piece of SAS program code to carry out the calculations.

HOW TO MANAGE THIS EXERCISE

View the results from the menu on the left. For the Province'91 population, four separate systematic samples of eight elements (q=N/n=32/8=4) are drawn from the population data set. The total of the study variable UE91 (number of unemployed in the province) is to be estimated.

In systematic sampling, a certain design parameter exists. This parameter, called the intra-class correlation coefficient rhoint, will be included in the design variance Vsys of an estimator of the population total. The magnitude of the intra-class correlation, and consequently its effect on variance estimates, depends partly on whether there is a successive system of ordering of the study variable values in the population frame. The effect of sorting order is examined in Part B of this training key.

To calculate the intra-class correlation we first decompose the population variance into the variation between the systematic samples and the variation within the systematic samples, as in standard one-way analysis of variance. In ANOVA terms we have SST = SSW + SSB, where SST represents the total sum of squares, SSW the within sum of squares and SSB the between sum of squares. By using the decomposition of the total sum of squares, the intra-class correlation is defined as
rhoint = 1 - n/(n-1)*SSW/SST.

Systematic sampling compared with simple random sampling without replacement is i) more efficient, if -1/(n-1) < rhoint < 0, ii) equally efficient, if rhoint = 0 or iii) less efficient, if 0 < rhoint < 1.

In this exercise the intra-class correlation is negative (-0.08) which means that given the current sorting order of the population, systematic sampling will be more efficient than simple random sampling without replacement (SRSWOR). Note that the population was pre-sorted first by the variable URB85 (urbanicity) and within each URB85 class, by the variable Municipality for this exercise.

The design effect is calculated by equation (2.24):
DEFFsys = Vsys(t_hat)/Vsrs(t_hat) = 1 + ( n-1)*rhoint. In this exercise the design effect is DEFF=0.43 which implies that systematic sampling is very efficient in this case.

Further, we can write the design variance of the total estimator in the form
Vsys(t_hat) = N2 * (1 - n/N) * S2/n * [1 + (n-1)*rhoint] or alternatively
Vsys(t_hat) = Vsrs(t_hat)*[1 + (n-1)*rhoint], as is given in equation (2.23).
This also implies that systematic sampling is more efficient than SRSWOR when rhoint is negative.
