Training key 59: The role of auxiliary information in PPS sampling

INTERPRETATION OF RESULTS WHEN X IS USED AS A SIZE MEASUREMENT VARIABLE

 

X is an artificially created variable for pedagogical purposes. Variable X is defined as UE91 + 3000. While variable X is used as the size measurement variable, some problems occur. Please pay attention to the following points in the output:

1) Correlation between UE91 and X:

The correlation of UE91 and X is ρ=1. The correlation is perfect and strong positive correlation is one indicator of a good size measurement variable. However, as we noted before, strong correlation is not sufficient enough itself for efficient estimation in a PPS sample. In the sense of a strong correlation, X is a perfect auxiliary variable for this case study. However, the problems occur once we view the scatterplot of UE91 against X.

2) Scatterplot of UE91 against X:

There is a perfect linear trend observed from the scatterplot of UE91 against Y. This is due to the correlation which is 1.  This means that the regression UE91=α + β*X holds perfectly. The problems lies in the fact that α is large and using the usual interpretation of regression coefficients, it can be shown that if α is large, i.e. the regression line is intercepts the Y axis far from the origin, then SRSWOR can be more efficient than PPS. In this case α is not extremely large but large enough to cause problems.

By performing the Monte Carlo simulation study, we note that although PPSWOR estimator is approximately unbiased, standard error becomes quite large. This means a lack of accuracy in estimation in a PPS sample when this type of auxiliary variable is used.
